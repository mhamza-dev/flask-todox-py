How to Use

Creating & Managing
  • Add New Todo
    Click the "Add New" button in the top right corner to create a new task
  • Edit Existing Todo
    Use the pencil icon to modify title, description, or completion status
  • Delete Todo
    Click the trash icon to remove a task permanently
Task Management
  • Mark as Complete
    Edit a task and check the "Completed" box to mark it as done
  • Track Progress
    View completion status with color-coded badges (green for completed, red for pending)
  • Auto-Save
    All changes are automatically saved to the database in real-time
